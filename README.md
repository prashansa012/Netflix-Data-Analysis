# Netflix Movie Data Analysis

## Project Overview

This project focuses on analyzing a dataset of over 9,000 Netflix movies using Python, Pandas, and Matplotlib. The objective is to explore movie trends, identify audience preferences, and generate actionable insights through data analysis and visualization.

The project simulates a real-world business scenario where data-driven insights help content platforms make informed decisions regarding content production and audience engagement.

---

## Business Questions Addressed

1. What is the most frequent movie genre on Netflix?
2. Which genres receive the highest number of votes?
3. Which movie has the highest popularity and what is its genre?
4. Which movie has the lowest popularity and what is its genre?
5. Which year produced the highest number of movies?

---

## Dataset Information

* Source: Kaggle
* Records: 9,000+ Movies
* Format: CSV
* Features include:

  * Movie Title
  * Genre
  * Popularity
  * Vote Count
  * Release Date
  * Original Language
  * Overview

## Project Workflow

### 1. Data Collection

* Imported Netflix movie dataset into Pandas DataFrame.

### 2. Data Cleaning

* Checked dataset structure and data types.
* Handled missing values.
* Removed duplicate records.
* Prepared data for analysis.

### 3. Exploratory Data Analysis (EDA)

* Genre analysis
* Popularity analysis
* Vote count analysis
* Release year trend analysis
* Content distribution analysis

### 4. Data Visualization

Created visualizations using Matplotlib to identify:

* Most common movie genres
* Highest voted genres
* Popularity distribution
* Movie release trends by year
